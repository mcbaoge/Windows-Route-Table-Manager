Windows Route Table Manager (exe version)
Overview

The Windows Route Table Manager is a simple and powerful graphical application designed to manage and modify the routing table of your Windows machine. This tool allows you to view, add, delete, and modify IPv4 routes directly from the user-friendly interface. The software is compiled into an executable .exe file for convenience, making it easy for users to manage routes without having to interact with the command line.

This project is distributed as a precompiled executable and does not include the source code.

Features

View and manage your system's IPv4 route table.

Add, delete, and modify routes with ease.

Set default route priority by adjusting the metric.

User-friendly graphical interface built with PyQt5.

Provides a simple way to manage networking without using the command line.

Usage

The software will automatically detect and list all existing routes on your system.

You can add new routes, modify existing ones, or delete routes using the available options in the interface.

You can also change the default route priority (metric) by selecting the appropriate network interface.

For permanent routes, you can choose to make the changes persistent, so they will remain even after a reboot.
